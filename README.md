# E-Commerce System (Java)

## 📌 Overview

This project is a console-based E-Commerce System implemented in Java using Object-Oriented Programming concepts like inheritance and polymorphism. It simulates product browsing, shopping cart operations, and order processing.

---

## 🚀 Features

### 🛍️ Product Management

* Abstract Product class
* Categories:

  * Electronics
  * Clothing
  * Books
* Product-specific attributes and discounts

### 🛒 Shopping Cart

* Add products to cart
* Update quantity
* View cart with total calculation

### 📦 Order Management

* Generate order with unique ID
* Calculate GST (18%)
* Display order summary and confirmation

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts (Inheritance, Polymorphism)
* Collections (ArrayList)
* Console-based UI

---

## ▶️ How to Run

```bash
javac ECommerceSystem.java
java ECommerceSystem
```
